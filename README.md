# Aprendendo-git-github-
Este repositorio foi criado com o objetivo de aprender e praticar os conceitos básicos de Git e GitHub

##  Oque é Git?
Git é um sistema de controle de versões que permite acompanhar alterações em arquivos e colocar em projetos.

## Oque é GitHub?
GitHub é uma plataforma online para hospedar repositórios Git e facilitar a colaboração.

## Principais comandos Git
- git init
- git clone
- git status
- git add
- git commit -m "mensagem"
- git push
- git pull

## Obejtivo
Aprender versionamento de código e boas práticas de commits.
